# Grass Single With Proxy or No Proxy
# Node Extension 1.25x

Clone this Repo
```bash
git clone https://github.com/Aidal7/grass-single.git
```
Open Directory
```bash
cd grass-single
```
Install Dependence
```bash
npm install
```
Edit Your Proxy if you want to use
```bash
nano proxy.txt
```
Run Script
```bash
node index-normal.js
```
